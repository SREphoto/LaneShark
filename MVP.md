# MVP Definition (Minimum Viable Product)

## LaneShark Bowling

### The "Must Haves" (V1.0)

The following features are essential for the game to be considered "playable" and ready for release.

1. **Core Gameplay Loop**
    - [x] Throw mechanics (Position -> Aim -> Power).
    - [x] Basic Physics (Ball movement, Pin collision).
    - [x] Scoring System (Standard 10-pin rules).
    - [x] Game Over state.

2. **User Interface**
    - [x] Scoreboard (real-time updates).
    - [x] Main Menu.
    - [x] End Game Summary.

3. **Progression (Basic)**
    - [x] Local Storage saving.
    - [x] Simple XP/Level accumulation.
    - [x] Basic currency (Money).

### The "Nice to Haves" (V1.1+)

Features currently implemented or planned for immediate follow-up.

1. **Advanced Physics**
    - [x] Hook mechanics (Spin).
    - [x] Lane Oil patterns.
    - [ ] Variable Pin weight.

2. **RPG Elements**
    - [x] Shop System.
    - [x] Inventory/Equipment logic.
    - [ ] Skill Trees.

3. **Polish**
    - [x] Sound Effects.
    - [x] Screen Shake / Impact particles.
    - [ ] Replays.
